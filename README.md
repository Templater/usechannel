## Available Commands

```bash
# Prepare environment (pnpm only)
$ pnpm run pinstall

# Start the development server on http://{instance address}:3000/
$ pnpm run dev

# Preview project
$ pnpm run preview

# Generate production build
$ pnpm run build

# Check code (pnpm only)
$ pnpm run plint

# Fix code (pnpm only)
$ pnpm run plint:fix
```
